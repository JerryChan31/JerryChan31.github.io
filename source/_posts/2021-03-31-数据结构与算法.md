### 时间和空间复杂度的计算

递归情况的复杂度计算：大概考虑一下

例：斐波那契（无优化）的时间复杂度：O(2^n)



主定理（Master Theorem）

| 算法                         | 复杂度     |
| ---------------------------- | ---------- |
| Binary Search                | O(log n)   |
| Binary tree traversal        | O(n)       |
| Optimal sorted matrix search | O(n)       |
| Merge Sort                   | O(n log n) |
